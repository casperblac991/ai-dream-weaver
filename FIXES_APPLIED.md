### 1. إصلاحات نظام المصادقة (Authentication)
- **المشكلة**: استخدام SHA256 بسيط لتشفير كلمات المرور (غير آمن)
- **الحل**: تحديث `app/auth.py` لاستخدام bcrypt
- **الملفات المعدلة**: `app/auth.py`
- **التحسينات**:
  - استخدام `passlib.context.CryptContext` مع bcrypt
  - إضافة دالة `verify_password()` للتحقق الآمن من كلمات المرور
  - تحديث دالة `login_user()` لاستخدام التحقق الآمن

## الخطوات التالية المطلوبة:

### 1. إضافة متغيرات البيئة
```bash
export TELEGRAM_TOKEN="your_token_here"
export API_URL="https://aidreamweaver.store"
export GROQ_API_KEY="your_groq_key"
export OPENAI_API_KEY="your_openai_key"
```

### 2. تشغيل البوتات
```bash
# بوت المدونة اليومي (يومياً)
python3 daily_blog_bot.py

# بوت تيليجرام (مستمر)
python3 all_bots.py

# بوت الحضارات (يومياً)
python3 civilization_bot.py
```

### 3. تشغيل المنصة الرئيسية
```bash
python3 app/main.py
# أو
uvicorn app.main:app --host 0.0.0.0 --port 10000
```
