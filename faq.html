ما هي منصة Weaver نَسَّاج؟+

Weaver (نَسَّاج) منصة متكاملة لتفسير الأحلام بالذكاء الاصطناعي تجمع بين تراث الحضارات القديمة (مصر، بابل، يونان، الإسلام) وأحدث تقنيات الذكاء الاصطناعي Groq + Llama 4 لتقديم تفسيرات دقيقة وعميقة، مع توليد صور فنية 4K مستوحاة من حلمك.

كيف يعمل التفسير بالذكاء الاصطناعي؟+

عند إرسال وصف حلمك، يقوم نظامنا بتحليله باستخدام نموذج Llama 4 عبر Groq API — وهو نموذج مُدرَّب على تفسيرات ابن سيرين والنابلسي والموروث الإسلامي والحضارات القديمة. يراعي النظام السياق والمشاعر ورموز الحلم للوصول إلى تفسير متكامل.

هل التفسير يعتمد على منهجية إسلامية؟+

نعم. يستند النظام إلى منهجية ابن سيرين (654–728م) ويعتمد على رمزية القرآن الكريم واللغة العربية في التفسير. مع ذلك، التفسير للاستئناس والتأمل وليس فتوى دينية.

ما الفرق بين Weaver والمواقع الأخرى؟+

Weaver هو الوحيد الذي يجمع: تفسير بالذكاء الاصطناعي + توليد صور 4K + موسوعة الحضارات القديمة + دعم كامل للعربية والإنجليزية. لا يوجد منافس عربي يملك هذه المزيج.

💰 الاشتراكات والأسعار
ما الفرق بين الخطط الثلاث؟+

• أساسي ($4.99/شهر): 5 تفسيرات + 3 صور يومياً
• محترف ($9.99/شهر): تفسير غير محدود + صور 4K غير محدودة + 50 أسلوب فني
• فريق ($19.99/شهر): كل مميزات المحترف + توليد فيديو + تقارير PDF + وصول API

كيف يتم الدفع؟+

الدفع عبر منصة Gumroad الآمنة (Visa, Mastercard, PayPal). بعد الاشتراك تصلك رسالة تأكيد بتعليمات التفعيل.

هل يمكن إلغاء الاشتراك؟+

نعم، يمكن الإلغاء في أي وقت عبر لوحة تحكم Gumroad بدون رسوم. يبقى وصولك فعالاً حتى نهاية الفترة المدفوعة.

هل هناك نسخة مجانية؟+

يمكنك تجربة البوت مجاناً عبر تيليجرام للحصول على فكرة عامة. للتفسيرات التفصيلية والصور 4K تحتاج لأحد خطط الاشتراك.

🤖 البوت والميزات
كيف أستخدم بوت تيليجرام؟+

1. افتح @aidreamweaver_bot على تيليجرام
2. اضغط Start أو أرسل /start
3. اكتب وصف حلمك بالعربية أو الإنجليزية
4. انتظر بضع ثوانٍ — ستصلك التفسير والصورة

ما جودة الصور المولَّدة؟+

الصور تُولَّد باستخدام نماذج SiliconFlow المتقدمة بدقة 1024×1024 (أساسي) و4K فنية (محترف). جودة سينمائية مستوحاة من رموز حلمك.

هل البوت يدعم العربية كاملاً؟+

نعم، البوت يفهم العربية الفصحى والعامية ويرد بالعربية، كما يدعم الإنجليزية بشكل كامل. يترجم البوت تلقائياً لتوليد الصورة ثم يرسل التفسير بلغتك.

🔒 الخصوصية والأمان
هل أحلامي خاصة وسرية؟+

نعم. لا نشارك أحلامك أو بياناتك مع أطراف ثالثة. تُستخدم البيانات فقط لتحسين جودة التفسير. راجع سياسة الخصوصية لمزيد من التفاصيل.
