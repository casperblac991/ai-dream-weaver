## 🚀 كيفية التشغيل

### 1. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 2. تعيين متغيرات البيئة
```bash
export TELEGRAM_TOKEN="your_token_here"
export API_URL="https://aidreamweaver.store"
export GROQ_API_KEY="your_groq_key"
export OPENAI_API_KEY="your_openai_key"
```

### 3. تشغيل المنصة الرئيسية
```bash
python3 app/main.py
# أو
uvicorn app.main:app --host 0.0.0.0 --port 10000
```

### 4. تشغيل البوتات (اختياري)
```bash
# بوت المدونة اليومي
python3 daily_blog_bot.py

# بوت تيليجرام
python3 all_bots.py

# بوت الحضارات
python3 civilization_bot.py
```
